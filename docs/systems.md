# Systems Overview

## Core Loop
1. Wybierz państwo.
2. Uruchom focus narodowy.
3. Rozdziel produkcję i badania.
4. Przemieszczaj armie po prowincjach.
5. Prowadź wojnę i zawieraj pokój.

## Systemy
- **Czas**: 1 sekunda realna = 1 godzina w grze, pauza i przyspieszenia x1/x2/x4.
- **Mapa**: siatka prowincji 12x10. Tryby mapy: polityczny, infrastruktura, zaopatrzenie, opór, lotnictwo.
- **Dyplomacja**: relacje liczbowe, napięcie światowe, pakty i sojusze (prototyp).
- **Produkcja**: kolejki sprzętu (prototyp).
- **Badania**: lista technologii i sloty badań.
- **Logistyka**: wartość zaopatrzenia z infrastruktury.
- **Lotnictwo/Marynarka**: uproszczone systemy misji i konwojów.
- **Eventy/Decyzje**: system wyboru opcji, koszt PP, cooldowny.
- **Armie**: ruch między prowincjami, bitwy i przejmowanie kontroli.

## AI
AI używa priorytetów focusów i prostych wag produkcji (do rozbudowy).
