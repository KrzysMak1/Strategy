# Known Issues (Audit Startowy)

Checklist największych braków po szybkim audycie repo:

- [ ] Brak spójnego banneru błędów w UI dla problemów systemowych (poza konsolą).
- [ ] Map modes istnieją, ale nie ma jasnych legend/tooltipów dla kolorów i statusów.
- [ ] Eventy uruchamiane losowo, bez logiki kontekstowej (wojna, daty, stany).
- [ ] AI nie reaguje na supply/produkcję, brak priorytetów frontu i logistyki.
- [ ] BattleSystem pomija teren, pogodę, umocnienia i role attacker/defender.
- [ ] SupplySystem jest uproszczony (brak hubów, kosztów propagacji).
- [ ] ProductionSystem nie ma efficiency i brakuje podziału na kategorie sprzętu.
- [ ] Research/Focus listy nie mają filtrów i jasnych opisów efektów.
- [ ] UI paneli (Focus/Research/Production/etc.) brak pełnej ergonomii na małych ekranach.
- [ ] Scenariusze nie nadpisują właścicieli prowincji ani armii (brak startowych override'ów mapy).
- [ ] Tryb step-day (debug) nie jest jeszcze dostępny w UI.
